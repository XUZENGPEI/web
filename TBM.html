三、TBM选型方法

方法一：规范阈值法（SL/T839-2025）

针对A₄超硬岩、B₄岩爆、C₄大变形、D₄构造破碎带4项按比例评分
≥70 → 敞开式 60～70 → 两者均可 <60 → 护盾式

方法二：多因素适宜性指数法

综合六大因素加权计算：岩体质量 · 完整性 · 磨蚀性 · 岩爆 · 大变形 · 突涌水
≥70 → 适宜 60～70 → 可行 <60 → 不适宜

权重：岩爆0.35 + 大变形0.35 + 构造破碎带0.2 + 超硬岩0.1（方法一）；方法二按两两对比构建六阶矩阵确定权重。

六、不良地质问题及应对措施

◈岩爆应对

护盾结构采用Q460高强合金钢；重载双排主推滚子轴承；微震监测系统（24h不间断）；超前释放地应力（钻孔+喷水）；涨壳式预应力中空锚杆+纤维喷砼+钢拱架联合支护。

◈软岩变形应对

刀盘与盾体偏心设计（顶间隙增大）；台阶式盾体（前大后小递减）；最大扩挖200mm；盾壳减阻注浆（摩阻系数降至0.15以下）；大推力大扭矩配置；护盾卡机预警系统（超声波+压力传感器）。

◈突泥涌水应对

总排水能力≥940m³/h；应急挡渣封堵装置；超前地质预报（TEAM‑2000电法系统）；超前钻孔注浆封堵；应急避险舱（25人96小时）；三级排水体系。

◈有害气体应对

实时监测CH₄/H₂S/CO/CO₂/NO₂/O₂等；CH₄达0.3%声光报警，0.5%断电；防爆部件配置；2.2m直径风管强通风（工作面风量1232m³/min）。

◈断层破碎带

规模断层（≥30m宽）采用旁通导洞钻爆法预处理，TBM步进/顶进通过；超前大管棚（φ108，L=20～30m）+超前锚杆；固结灌浆L=4.5m。

◈高地温处理

中高温带（Ⅱ类，37～49℃）采用人工制冷；低高温带（Ⅰ类）加强通风；TBM设备配置水冷系统。

七、支护设计要点

护盾式TBM（双护盾/单护盾）支护

C50～C80钢筋混凝土管片，厚55cm，环宽1.5～1.8m。
管片分6块（含仰拱块、封顶块、邻接块、标准块）。
矩形分块，错缝拼接。M30弯螺栓连接（环向12个+纵向16个）。
管片背后充填20cm豆粒石+回填灌浆。

敞开式TBM支护

"喷锚+挂网钢筋+工字钢"联合支护。
Ⅲ类：喷C30W8砼12cm + 锚杆L=3.5m
Ⅳ类：厚18cm + 锚杆L=4.0m + H125@1.8m钢支撑
Ⅴ类：厚20cm + 锚杆L=4.5m + H150@0.9m + 超前大管棚L=20m
变形段：聚丙烯纤维砼 + 预应力锚索 + HW200钢支撑

  1. TBM‑1（二郎山进口段·10.52km）：岩性以硬岩为主，围岩以Ⅲ类为主，强岩爆洞段占比高。考虑强岩爆的施工安全性，推荐双护盾TBM。
  2. TBM‑2（二郎山出口段·9.77km）：包含硬岩与页岩等软岩，断层破碎带、突涌水、岩爆等。考虑快速掘进及复杂不良地质处理灵活性，推荐敞开式TBM（配置钢管片安装功能）。
  3. TBM‑3（老君山隧洞·9.15km）：岩性均为硬岩，以Ⅲ类围岩为主（69%），不良地质规模不大。从处理灵活性和经济性，推荐敞开式TBM。
  4. TBM‑4（莲花山前段·14.96km）：以中硬岩为主，围岩以Ⅲ类为主。从不良地质处理灵活性和经济性，推荐敞开式TBM。
  5. TBM‑5（莲花山后段·15.99km）：以软岩为主（Ⅴ类占60.2%），存在软岩变形、高瓦斯。从TBM适应性和施工效率，推荐单护盾TBM。
